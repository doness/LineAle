# LineAle
_May the Ale be with you..._

Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone http://github.com/doness/LineAle

Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run Ale :

    $ python alebot.py
